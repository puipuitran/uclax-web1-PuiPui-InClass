01. Explain the difference between HTML, CSS and JS.

HTML: HTML stands for Hypertext Markup Language. It creates the structure of a website. HTML requires developers to use a predefined list of tags. HTML is static - meaning items are not animated. CSS: CSS stands for Cascading Stylesheets. It allows developers to determine the size, color, and locations of the items on the website. HTML is static - meaning items are not animated. JavaScript: Javascipt gives you the ability to change the HTML and CSS on the page. It allows developers to create interactions on the website.
